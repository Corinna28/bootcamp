Exercice 06 : Les boucles

Énoncé :

Écrire un programme Javascript qui demande à l’utilisateur de rentrer un nombre.
Ensuite ce nombre devra être utilisé pour affiché sa table de multiplication et

Il devra être affiché dans la console comme suit :

  • jusqu'à....
  • 7 x 10 = 70
