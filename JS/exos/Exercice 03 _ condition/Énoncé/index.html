Exercice 03 : Les conditions

Énoncé :

Écrire un programme Javascript qui demande le mail et le mot de passe de l’utilisateur, qu’on aura au préalable définit dans notre script.

Vérifier la condition suivante :

  • Si l’email et le mot de passe ne correspondent pas aux valeurs définies, le message d’alerte devra afficher «Identifiants incorrects»
  • Sinon, le message d’alerte devra afficher «Bienvenue dans votre espace client»
